# Customer-Behavior-Analysis
End-to-end retail analytics project using Python, SQL, and Power BI.

# 🛍 Customer Shopping Behavior Analysis  
### End-to-End Data Analytics Project

---

## 📌 Business Problem

A retail company wanted to understand customer shopping behavior to improve sales, customer satisfaction, and long-term loyalty.

Key Question:
> How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing strategies?

---

## 📊 Dataset Overview

- 3,900 customer transactions  
- 18 columns  
- Demographics, purchase details, discounts, subscription status  
- 37 missing values handled in Review Rating column  

---

## 🛠 Tech Stack

- Python (Pandas, NumPy)
- PostgreSQL
- SQL
- Power BI
- CSV Dataset

---

## 🔄 Project Workflow

### 1️⃣ Data Cleaning & EDA (Python)

- Loaded CSV dataset
- Handled missing values
- Standardized column names
- Created new features (age_group, purchase_frequency)
- Removed redundant columns
- Exported cleaned data to PostgreSQL

---

### 2️⃣ Business Analysis (SQL)

Performed structured SQL analysis to answer business questions:

- Revenue by Gender
- High-Spending Discount Users
- Top 5 Products by Rating
- Shipping Type Comparison
- Subscribers vs Non-Subscribers
- Customer Segmentation (New, Returning, Loyal)
- Revenue by Age Group

---

### 3️⃣ Dashboard Development (Power BI)

Created an interactive dashboard showing:

- Total Customers
- Average Purchase Amount
- Revenue by Category
- Revenue by Age Group
- Subscription Insights
- Sales Trends

---

## 📈 Key Insights

- Loyal customers represent majority of transactions.
- Young Adults generate highest revenue.
- Discount-heavy products drive volume.
- Express shipping customers spend slightly more.
- Subscription users show strong repeat behavior.

---

## 💡 Business Recommendations

- Improve subscription conversion strategies.
- Strengthen loyalty programs.
- Optimize discount policies to protect margins.
- Focus marketing on high-revenue age groups.
